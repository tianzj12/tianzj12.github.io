文件系统目录

Linux里一切皆文件

由根目录 / 下分支出以下主要目录

/bin 存放经常使用的命令

/sbin 系统管理员使用的系统管理程序

/home 存放普通用户的主目录

/root 系统管理员（超级权限者）的用户主目录

/etc 所有的系统管理员所需要的配置文件和子目录

/usr 用户的许多应用程序和文件都存在这个目录下

/boot 启动Linux时使用的一些核心文件

/media 把当前识别的设备挂载到该文件夹下

/mnt 临时挂载别的文件系统（共享文件夹在该目录下）

/var 经常被修改的文件存储在这里

/opt 额外安装软件所摆放的目录（安装包）

/usr/local 额外安装软件所安装的目录（安装好的程序）

/lib 系统开机所需要的最基本的动态连接共享库

/proc（不要动） 系统内存的映射

/srv（不要动） 存放一些服务启动后需要提取的数据

/sys（不要动） 文件系统

/tmp 存放临时文件

/dev 将硬件以文件方式存储

/selinux 安全子系统，能控制程序只能访问特定文件

/lost+found 系统非法关机后存放文件

vim编辑器

快捷键

  • yy 拷贝当前行，5yy 拷贝当前行向下的5行（可替换为任意行），p 粘贴
  • dd 删除当前行，5dd 删除当前行向下的5行（可替换为任意行）
  • / 文档中查找某单词，n 查找下一个单词
  • ：set nu 添加文件的行号，set nonu 取消文件的行号
  • G 到达文档最末行，gg 到达文档最首行（正常模式下指令）
  • u 撤销（正常模式下指令）
  • 数字+shift+g 光标移动至指定行

    开关机重启等操作

shutdown

  • shutdown -h now 立即关机
  • shutdown -h 1 1min后关机
  • shutdown -r now 立即重启

halt
直接使用，关机

reboot
重启

sync
将内存中的数据同步到磁盘上（良好习惯）

用户登录和注销

普通用户登录后可以使用“su - 用户名”切换为系统管理员身份，输入logout命令即可注销用户

运行级别

存储在系统的运行级别配置文件/etc/inittab中（7个运行级别）

0：关机

1：单用户（找回丢失密码）

2：多用户无网络

3：多用户有网络（常用）

4：保留的待开发运行级别

5：图形界面（常用）

6：重启

基本语法

  • init [运行级别] 切换至指定运行级别

📑 如何找回root密码
思路：切换至运行级别1，重设密码
（开机 -> 引导处Enter -> 按e -> 选中第二行（编辑内核）输入e -> 输入1+Enter -> 输入b -> 单用户模式 -> 使用 passwd 指令修改root密码）

实用指令

帮助指令

man [指令名称] 获取指令的帮助信息

help [指令名称] 获取shell内置命令的帮助信息

文件目录类

  • pwd 显示当前工作目录的绝对路径

  • ls [选项] [目录 or 文件] 查看当前目录中的信息（常用选项 -a：显示包括隐藏的当前目录下所有文件和目录；-l 以列表形式显示信息）

  • cd [参数] 切换至指定目录

      cd or cd~    回到 /root（用户家目录）
  cd..    回到上一级目录

  绝对路径：/home    从根目录开始定位
  相对路径：../home    从当前目录开始定位

  • mkdir [选项] [目录路径] 在指定位置创建目录（-p：创建多级目录）

  • rmdir [选项] [目录路径] 删除指定位置的目录

  • rm [选项] [目录路径] 删除指定位置的目录（-r：递归删除整个文件夹；-f：强制删除不提示）（常用 rm -rf 删除整个文件夹）

  • touch [选项] [文件名称] 指定位置创建文件（可创建多个，用空格间隔）

  • cp [文件名称] [目录名称] 将文件拷贝至指定目录下
    cp -r [目录名称1] [目录名称2] 将目录1全部内容递归复制至目录2位置
    \cp -r [目录名称1] [目录名称2] 将目录1全部内容递归复制至目录2位置，直接覆盖目录2下相同文件

  • mv 移动文件或重命名
    mv [文件名] [路径] 将文件移动至指定路径
    mv [文件名1] [文件名2] 将当前目录下的文件1重命名为“文件名2”

  • cat [选项] [文件名称] 以只读方式查看文件（-n：显示行号）
    cat [选项] [文件名称] | more 以只读方式查看文件，并分页显示（常见用法）
    cat [文件1] > [文件2] 将文件1内容覆盖写入到文件2中

  • more [文件名称] 分页显示文件（按 Enter 逐行查看；按 Space 逐页查看；Ctrl + B 上一页；Ctrl + F 下一页）

  • less [文件名称] 分页显示文件，但不是一次性加载全部内容，对于大型文件效率高

  • [显示内容的指令] > [文件名] 重定向指令：将前面的内容覆盖写入到指定文件中（若无该文件则创建文件）

  • [显示内容的指令] >> [文件名] 追加指令：将前面的内容追加写入到指定文件中

  • echo “[内容]” [> or >>] [文件名] 将前面的内容覆盖（or 追加）写入到指定文件中
    echo [选项] [输出内容] 输出指定内容

  • head [-n 数字] [文件] 输出文件的前 [数字] 行（默认10行）

  • tail [-n 数字] [文件] 输出文件的前 [数字] 行（默认10行）
    tail -f [文件]：实时追踪该文档的所有更新，常用

  • ln -s [原文件 or 目录] [软链接名称] 为目标创建一个软链接（快捷方式）

  • history 显示所有执行过的指令
    history [数字] 显示后 [数字] 行执行过的指令

  • ![数字] 执行编号为 [数字] 的指令

时间日期类

  • cal 显示日历
    cal [年份] 显示 [年份] 年的全部日历
  • date 显示当前时间
    date +%Y 显示当前年份
    date +%m 显示当前月份
    date +%d 显示当前是哪一天
    date “+%Y-%m-%d %H:%M:%S” 显示年月日时分秒（格式字符串）
    date -s [字符串时间] 将系统时间设置为 [字符串时间]

搜索查找类

  • find [搜索范围] [选项] [文件名] 在 [搜索范围] 内按 [选项] 搜寻 [文件名] 对应的文件
    （例1：find /home -name hello.txt 查找/home下名称为”hello.txt”的文件）
    （例2：find /home -name *.txt 查找/home下所有 txt 文件）
    （例3：find /home -size +20M 查找/home下大于20M的文件）
  • locate [搜索文件] 快速定位文件，无序遍历整个文件系统，首次使用前需要使用 updatedb 指令创建 locate 数据库
  • grep 过滤查找（-n：显示匹配行及行号；-i：忽略字母大小写）
    常见用法：cat [文件] | grep [查找内容] 显示 [文件] 中的 [查找内容]
  • [命令1] | [命令2] 管道符号”|”：将命令1的处理结果交给命令2进行处理

压缩和解压缩

  • gzip [文件] 将文件压缩为 .gz 文件（不保留原文件）
  • gunzip [文件.gz] 将文件.gz 解压缩
  • zip [选项] [压缩文件名.zip] [待压缩文件] 将文件压缩为 .zip 文件（-r：递归压缩目录）
  • unzip [待解压压缩包.zip] 将文件.zip 解压缩（-d [目录]：指定解压后文件存放位置）
  • tar [选项] [压缩包名称.tar.gz] [文件] 将 [文件] 解压 or 压缩（-c：产生.tar.gz 打包文件；-v：显示详细信息；-f：指定压缩后的文件名；-z：打包同时压缩；-x：解压.tar.gz 文件）

其他指令

  • Tab 补齐
  • Ctrl+C 终止程序运行
  • Ctrl+Z 挂起当前进程
  • Ctrl+/ 终止前台进程并生成 core 文件
  • Ctrl+D 表示一个特殊的二进制值，表示 EOF
  • Ctrl+L 清空窗口
